# Brainbox Candy — Trade CRM

Custom CRM for managing trade customer relationships.

## Features
- Customer directory synced from KashFlow
- Contact event timeline per customer
- Bulk CSV event logging
- MailerLite campaign tracking
- Team accessible via web browser

## Tech Stack
- Frontend: React + Vite + Tailwind CSS
- Backend: Supabase (Postgres)
- Auth: Supabase Auth
- Hosting: TBD
